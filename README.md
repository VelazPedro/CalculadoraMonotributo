# CalculadoraMonotributo
Calculadora de monotributo en C. Proximamente, su versión en Python con acceso a una API que la mantenga al día.
Se puede ejecutar sin muchas complicaciones tan solo con su .exe.
Para ejecutarse, basta con reemplazar la tabla en uso en la función TABLA con la nueva tabla proporcionada por ARCA en cada recategorización.
